## 📊 交互流程

### 用户操作流程
1. 点击首页左侧边栏的用户头像
2. 弹出登录对话框
3. 选择登录方式（验证码/密码）
4. 填写表单信息
5. 勾选用户协议
6. 点击登录按钮
7. 执行登录逻辑（日志输出）
8. 关闭对话框
