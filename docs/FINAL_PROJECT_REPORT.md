## 🏗️  架构特点

### 分层架构
```
lib/
├── features/auth/           # 认证功能
│   ├── domain/             # 领域层
│   │   ├── entities/       # 实体
│   │   ├── repositories/   # 仓库接口
│   │   └── services/       # 服务接口
│   └── data/               # 数据层
│       ├── repositories/   # 仓库实现
│       └── services/       # 服务实现
├── pages/                  # UI页面
├── widgets/                # 共享组件
└── state/                  # 状态管理
```

### 技术栈
- **框架**: Flutter 3.5+
- **设计**: Material Design 3
- **状态管理**: Provider
- **本地存储**: SharedPreferences
- **架构模式**: Clean Architecture
