## 🔄 页面流程

### 默认行为
1. 应用启动 → 显示首页（Home图标Tab）
2. 左侧边栏默认选中"模型库"
3. 主内容区显示项目网格（12个项目卡片）

### 交互
1. 点击侧边栏菜单 → 切换主内容区
2. 点击顶部Tab → 切换不同页面
3. 点击项目卡片 → 打开项目（日志输出）
4. 点击"创建项目"/"打开项目" → 触发操作（日志输出）
